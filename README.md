Hey, I'm Shreya 👋
I started out in electronics — writing firmware, wiring up sensors, building things that blink and buzz. Somewhere along the way I got more interested in what the data meant than what the hardware did. So I made the jump.
Now I'm finishing up my Master's in Business Analytics & Big Data at IE Business School in Madrid, and I spend most of my time working with messy real-world datasets, building pipelines, and turning numbers into something a non-technical person can actually act on.
I like problems that have stakes. Infrastructure, elections, retail decisions — the kind of work where bad analysis has real consequences.

What I've been working on
🌍 Seismic Risk Analysis Pipeline
Built an end-to-end data pipeline to process USGS seismic sensor data — ingestion via Apache NiFi, storage in MinIO, distributed processing in Spark, and K-means clustering to identify high-risk zones. The interesting part was making it robust enough to handle the messiness of real sensor feeds.
Python · Apache Spark · NiFi · MinIO · scikit-learn

⚡ IE × Iberdrola Sustainability Datathon
Team project. My piece was analysing Viesgo grid capacity across northern Spain — Cantabria, Asturias, Galicia — to figure out where the bottlenecks are for the 2027 national EV charging rollout. Found that 77% of substations in that corridor have under 2 MW available capacity, median was basically 0. That's a real infrastructure problem.
Python · GeoPandas · pyproj · Spark

🛒 Retail Customer Insights
Customer segmentation and behaviour analysis on retail transaction data — RFM analysis, clustering, churn indicators. Goal was to give the business actionable segments, not just pretty charts.
Python · pandas · scikit-learn · Tableau

👁️ Vision Mate
This one goes back to my hardware days. An assistive navigation device for visually impaired users — obstacle detection, GPS tracking, real-time alerts. Built on ESP32 + Arduino, with sensor data streaming through ThingSpeak. I think of it as my first real IoT data pipeline, I just didn't call it that at the time.
ESP32 · Arduino · GPS · ThingSpeak · IoT

📊 Tata Retail Customer Analysis
Analysed sales and customer data for a retail case study, built dashboards to surface revenue trends, regional performance, and customer lifetime value. Focused on making the insights self-explanatory for a non-analyst audience.
Tableau · Python · Excel

🗳️ District Elections Dashboard (Internship — Hareesh IT Industries)
Built during my data analyst internship. Interactive Power BI dashboard visualising Indian state-level election results — voter turnout, seat distribution, party performance across districts. One of the first times I saw a stakeholder actually change their opinion based on something I built.
Power BI · SQL · Excel
